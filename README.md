## Jumpserver Docker-Compose

- 全面支持 arm64 国产化操作系统

系统架构
- [x] Linux/amd64
- [x] Linux/arm64

环境要求
- MySQL Server >= 5.7
- Redis Server >= 6.0

> 请先自行创建 数据库 和 Redis, 版本要求参考上面环境要求说明

```sh
git clone --depth=1 https://github.com/wojiushixiaobai/Dockerfile.git
cd Dockerfile
cp config_example.conf .env
vi .env
```
